# BUILD & RUN FRONT END MELI CHALLENGE PROYECT!

depedencies to install

- Node v20.11.0 
- npm 10.2.4
- react v18.2.0

To deploy and run melichallenge_be_me proyect you should type on your terminal:

    $ git clone https://github.com/EVELYN2717/melichallenge.git
    $ cd melichallenge
    $ npm i
    $ npm start
